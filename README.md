- 🌐 **Enterprise Networking**  
  Designing and troubleshooting enterprise-grade network topologies using routing and switching technologies.

- 🔐 **Network Security**  
  Exploring firewall concepts, ACLs, segmentation, and security monitoring techniques used in enterprise environments.

  Learning redundancy, failover, WAN connectivity, and high availability concepts used in production networks.

- 🐍 **Python for Network Automation**  
  Exploring scripting and automation for network operations, monitoring, and security workflows.

Recently, I’ve been building hands-on networking labs and enterprise-level simulations to strengthen my practical understanding of real-world infrastructure and troubleshooting scenarios.

My long-term goal is to grow into a highly skilled Network / Security Engineer capable of designing, securing, and maintaining large-scale enterprise environments.
